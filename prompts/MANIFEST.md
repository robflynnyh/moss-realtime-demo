# Prompt Voice Samples

Run `scripts/download_prompt_samples.sh` to download the source Ogg files and
trim 12-second mono 24 kHz WAV prompts into this directory.

The first group is public-domain Wikimedia Commons speech recordings from
deceased U.S. presidents:

| File | Source | License note |
| --- | --- | --- |
| `jfk_berlin_12s.wav` | <https://commons.wikimedia.org/wiki/File:Jfk_berlin_address_high.ogg> | Wikimedia lists it as public domain in the U.S. as a U.S. Federal Government work. |
| `nixon_resignation_12s.wav` | <https://commons.wikimedia.org/wiki/File:Nixon_resignation_audio_with_buzz_removed.ogg> | Wikimedia lists it as public domain in the U.S. as a U.S. Federal Government work. |
| `fdr_fireside_12s.wav` | <https://commons.wikimedia.org/wiki/File:FDR_Chat_Mar_37.ogg> | Wikimedia lists it as public domain in the U.S. as a U.S. Federal Government work. |

The second group is drawn from Nabu Casa/OHF-Voice public voice datasets, which
the repository describes as CC0 public-domain voice datasets:

| File | Source | License note |
| --- | --- | --- |
| `nabu_joe_en_us_12s.wav` | <https://github.com/OHF-Voice/voice-datasets/tree/master/en_US> | CC0/public domain per the OHF-Voice repository README. |
| `nabu_kathleen_en_us_12s.wav` | <https://github.com/OHF-Voice/voice-datasets/tree/master/en_US> | CC0/public domain per the OHF-Voice repository README. |
| `nabu_kerstin_de_de_12s.wav` | <https://github.com/OHF-Voice/voice-datasets/tree/master/de_DE/kerstin> | CC0/public domain per the OHF-Voice repository README. |
| `nabu_dave_es_es_12s.wav` | <https://github.com/OHF-Voice/voice-datasets/tree/master/es_ES/dave> | CC0/public domain per the OHF-Voice repository README. |
